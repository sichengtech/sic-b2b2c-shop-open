操作表

操作表可以让用户从多个可选的操作中选择一个

$.actions(buttons) - 创建并弹出一个操作表

  • buttons - 显示的几个按钮，你也可以把按钮分组

你可以通过 color: "danger" 来指定按钮的文案为红色

或者通过 bg: "danger" 来指定按钮的背景为红色
